Introduction

HTML, or HyperText Markup Language, is a markup language used to describe the structure of a web page. It uses a special syntax or notation to organize and give information about the page to the browser. Elements usually have opening and closing tags that surround and give meaning to content. For example, there are different tag options to place around text to show whether it is a heading, a paragraph, or a list.

Cascading Style Sheets (CSS) tell the browser how to display the text and other content that you write in HTML. Note that CSS is case-sensitive so be careful with your capitalization. CSS has been adopted by all major browsers and allows you to control:

  • color
  • fonts
  • positioning
  • spacing
  • sizing
  • decorations
  • transitions

There are three main ways to apply CSS styling. You can apply inline styles directly to HTML elements with the style attribute. Alternatively, you can place CSS rules within style tags in an HTML document. Finally, you can write CSS rules in an external style sheet, then reference that file in the HTML document. Even though the first two options have their use cases, most developers prefer external style sheets because they keep the styles separate from the HTML elements. This improves the readability and reusability of your code.

Positioning

The positioning properties in CSS include the following:

Relative:

It allows to specify how CSS should move it relative to its default position. It is accompanied by the CSS offset properties of left or right, and top or bottom. These day how many pixels, percentage, or ems to move the item away from its default/normal position.

For example: I want to move the h2 10 pixels away from the bottom and 20 pixels away from the top:

        
    h2 {
    position: relative;
    top: 20px;
    bottom: 10px;}

Absolute:

Locks the element relative to its parent container. It's important to add a position rule to the parent element (by using position: relative in the parent element), otherwise, the browser will keep looking up the chain and ultimately will use the body tag position. This position is also accompanied by the CSS offset properties of left or right, and top or bottom.

For example: I want to lock the h2 50 pixels away from the top and from the right from its parent element (in this case a div class named container):

        
        .container {
    position: relative;} 
        h2 {
    position: absolute;
    top: 50px;
    bottom: 50px;}

Fixed:

Locks the element relative to the browser window An element with a fixed positioned won't move when the user scrolls.

For example: I want the navigation bar (with a class named navbar) from my page to keep showing top when I scroll the page:

        
        .navbar {
    position: fixed;
    top: 0px;
    right: 0px;}

Other positioning uses are the following:

Z-index:

Used to change the position of overlapping elements. It can specify the order of how elements are stacked on top of one another.

For example: I have two boxes, with classes .first and .second, if i want the .first box to cover the .second box, I would need to do the following:

        
        .first {
    position: absolute;
    z-index: 1; this means it will go in front of the other element}
    .second {
    position: absolute;}

Margin auto

A way to horizontally center an element is by using margin: auto.

For example: I want to move my box with class .box to the center of my page (horizontally):

        
        .box {
    margin: auto;}

Animation with CSS

There are 8 animation properties in total:

  • animation-name

    • As the name says, this animation property sets the name of the animation. The animation-name property will be used by @keyframes to tell CSS which rules go with each animation property (rules that control what happens during each animation). This is done by giving CSS properties for specific "frames" during the animation with percentages, being 0% the start of the animation and 100% the end.

  • animation-duration

    • This animation property sets the length of time for the animation. Here is an example using both animation-name and animation-duration properties: I want to create an animation with color changes to my element with class name .circle:

              
     .circle {
     animation-name: color-change;
     animation-duration: 5s;}

     @keyframes color-change {
     0% {
     background-color: pink;
     }

     50% {
     background-color: purple;}

     100% {    
     background-color: red;} }
  • animation-timing-function

    • Controls how quickly an animated element changes over the duration of the animation. The values are:

    • ease: default value, which starts slow, speeds up and then slows down in the end.
    • ease-out: quick at start and slows down
    • ease-in: slow at start and speeds up
    • linear: applies constant speed

    This property can also be controlled with Bezier curves, these are used with the cubic-bezier function. You control the animation with the four main points (p0, p1, p2, p3) being as follows (x1, y1, x2, y2).

  • animation-delay

    • This property specifies a delay for the start of an animation. The value is defined in seconds (s) or milliseconds (ms).

  • animation-iteration-count

    • This property allows you to control how many times you want your animation to repeat. For example, I want my .circle to repeat its animation continuously:

              
     .circle {
     animation-name: color-change;
     animation-duration: 5s;
     animation-iteration-count: infinite;}

     @keyframes color-change {

     50% {
     background-color: red;
     }
     100% {
     background-color: blue;}}
  • animation-direction

    • This property allows you to control whether your animation should be played forwards, backwards or in alternate cycles.

  • animation-fill-mode

    • This property used as: animation-fill-mode: forwards; keeps the last thing of animation on the element. For example: I want my .circle (that is initially pink), to keep the color blue while hovered:

              
     .circle:hover {
     animation-name: color-highlight;
     animation-duration:  3s;
     animation-fill-mode: forwards;}

     @keyframes color-highlight {
     100%  {
     background-color: blue;}}
  • animation-play-state

    • This property defines whether the animation is running or paused.

Responsive Web Design

Media Query

It's used to define how content is seen depending on type of device. Values can be, max-width, max-height, min-width, min-height. For example: I want my content to display in a certain way when the user is using a device with width less than 300px:

          
     @media (max-width: 300px) {
     /*CSS rules to be applied*/}

Responsive Images

To make an image responsive, you just need to add the following properties to an image:

          
      img {
      max-width: 100%
      height: auto;}

Retina Image

To make images optimized for High-resolution displays, you need to define their width and height values as half of what the original file is.

Responsive Text

Make a text responsive by using Viewport units (width or height) and percentages relative to its parent container size. The four viewport units are:

  • vw (viewport width): 10vw would be 10% of the viewport's width.
  • vh (viewport height): 3vh would be 3% of the viewport's height.
  • vmin (viewport minimum): 70vmin would be 70% of the viewport's smaller dimension (height or width).
  • vmax (viewport maximum): 100vmax would be 100% of the viewport's bigger dimension (height or width).

Flexbox

Placing the CSS property display:flex on an element allows you to use other flex properties to build a responsive page.

To align the "flexed element" into rows and columns, you add the flex-direction property to the parent item and set it to row or column.

main-axis cross-axis
for rows, the main axis is a horizontal line and for columns the main axis is a vertical line. for rows, the cross axis is a vertical line and for columns the cross axis is a horizontal line.

The following are properties for flex container:

Justify Content

To tell CSS how to align and space out the flex items within a container a certain way along the main-axis. (How they fill the space of the container).

  • center: aligns all the flex items to the center of the container.
  • flex-start: aligns the items to the start of the container, for columns, this pushes the items to the top of the container, and for rows, this pushes the items to the left of the container. (If there is no justify-content specified, this is the default alignment).
  • flex-end: aligns the items to the end of the container, for columns, this pushes the items to the bottom of the container, and for rows, this pushes the items to the right of the container.
  • space-between: aligns items to the center of the main axis, with extra space placed between the items. This pushes the first and last item to the very edge of the container.
  • space-around: similar to space-between, but the first and last item have a space between item and edge of container.
  • space-evenly: distributes space between flex items evenly, giving a full space at either end of the container.

Align-Items

To tell CSS how to align and space out the flex items within a container a certain way along the cross-axis. (How they fill the space of the container).

  • center: aligns items to the center. For rows, this vertically aligns items (equal space above and below the items). For columns, this horizontally aligns items (equal space to the left and right of the items).
  • flex-start: aligns the items to the start of the container, for rows, this pushes the items to the top of the container, and for columns, this pushes the items to the left of the container. (If there is no justify-content specified, this is the default alignment).
  • flex-end: aligns the items to the end of the container, for rows, this pushes the items to the bottom of the container, and for columns, this pushes the items to the right of the container.
  • stretch: stretch the items to fill the flex container. For example, rows items are stretched to fill the flex container top-to-bottom. This is the default value if no align-items value is specified.
  • baseline: aligns items to their baselines.

Grid

Once you set display:grid in your container, you can define the structure of the grid with both columns and rows. To add columns to the grid, use the grid-template-columns property on a grid container. For example, I want to set three columns on my container: *we specify the width in columns*

        
    .container {
    display: grid;
    grid-template-columns: 20px 20px 20px;}

This will set the number of rows automatically, according to the specified width and number of columns. To adjust the rows manually, use the grid-template-rows property in the same way. *we specify the height in rows*

Change size of Columns and Rows

To define the size of rows and columns you can use the following units:

  • px: pixels as we know
  • em: size according to parent element
  • fr: sets the column or row to a fraction of the available space
  • auto: sets the column or row to the width or height of its content automatically
  • %: adjusts the column or row to the percent width of its container

For example: I want to create five columns. The first column is as wide as its content (auto), the second column is 50px, the third column is 10% of its container, and for the last two columns, the remaining space is divided into three sections, two fractions of the available space are allocated for the fourth column, and one fraction is allocated for the fifth column:

        
    grid-template-columns: auto 50px 10% 2fr 1fr;

Add Gap in Grid

To add a space between columns, use grid-column-gap with pixels. To add space between tows, use grid-row-gap with pixels. To shorten this, you can use grid-gap, if you add one value, it will apply for both rows and columns, if you add two, the first one will apply to rows and second one to columns.

For example, I want to create a gap of 5px between my rows and 20px between my columns:

  
        
     grid-gap: 5px 20px;

Media Query in Grid for Responsiveness

Make your site responsive with Grid by using @media:

For example: I want to make the header area occupy the top row completely and the footer area occupy the bottom row completely when the viewport width is 400px or more.

        
      @media (min-width: 400px) {
      .container {
        grid-template-areas: 
          "header header"
          "advert content"
          "footer footer"}
